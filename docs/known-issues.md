# よくある問題と確認点

## ランキング登録できない

確認すること：

- Supabase URL
- Publishable key
- `game_slug`
- `submit_score` の引数
- RPCの戻り値
- ネットワークエラー表示

## Supabaseに登録したのに実験場に出ない

実験場側に `GAMES` 配列がある場合、Supabase登録だけでは表示されない。
実験場トップと詳細ランキングページの両方を確認する。

## score_scale不一致

秒や小数表示のゲームでは、内部整数と表示値がずれやすい。
`score_scale` と `score_decimals` を必ず確認する。

## 結果画面にランキングがない

仕様で結果画面下部ランキングが必要な場合、結果画面に表示場所を用意する。
ランキング0件でも表示が壊れないようにする。

## 名前入力が任意のまま

ランキング必須ゲームでは、名前入力を必須にする。
仕様で自動送信と決めたゲームでは、終了時に送信する。

## 二重送信

結果画面表示、リトライ、ホーム移動、連打で複数回送信されないようにする。

## スマホ操作の問題

- 長押しメニュー
- テキスト選択
- ダブルタップ拡大
- ピンチズーム
- ゲーム中スクロール
- 画面下部ボタンが押しにくい

## 古いコードが混ざる

古いコード参照のため、途中版は `index_v2.html`、`index_v3.html` のように名前を変える。
公開正本は常に `index.html`。
