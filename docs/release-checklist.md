# 公開前チェックリスト

## 仕様

- [ ] ゲーム名が正しい
- [ ] リポジトリ名が正しい
- [ ] game_slugが正しい
- [ ] ルール説明が現在仕様と一致している
- [ ] 古いタイトル、説明、ボタン文言が残っていない

## 画面

- [ ] ホーム画面がある
- [ ] ゲーム画面がある
- [ ] 結果画面がある
- [ ] ルール説明が読める
- [ ] 実験場リンクがある
- [ ] 結果シェアができる

## スマホ

- [ ] iPhone SE級で横スクロールしない
- [ ] ボタンが小さすぎない
- [ ] ゲーム中にページが不要に動かない
- [ ] 長押しコピーが出ない
- [ ] ダブルタップ拡大しない
- [ ] ピンチズームの扱いが仕様通り
- [ ] 画面下部のボタンが押しやすい

## ゲーム

- [ ] 開始できる
- [ ] 終了できる
- [ ] リタイアが必要なゲームでは動く
- [ ] 勝利、敗北、失敗、タイムアウトが処理される
- [ ] 理論上プレイ可能
- [ ] スコアが不自然に増減しない

## ランキング

- [ ] Supabase URLが正しい
- [ ] Publishable keyが入っている
- [ ] service_role keyが入っていない
- [ ] game_slugが一致している
- [ ] スコア送信が1回だけ
- [ ] 送信中、成功、失敗が表示される
- [ ] 結果画面下部にランキングが必要な場合、表示される
- [ ] 実験場トップと詳細ランキングで表示できる

## 公開

- [ ] 公開ファイル名は `index.html`
- [ ] 古い参照用ファイルは `index_v2.html` などに分けた
- [ ] 変更点を記録した
- [ ] 手動確認端末を記録した
